How to get your Facebook API credentials:

  1. Go to Facebook for Developers and log in with your Facebook account.
  2. Create a new app or select an existing one.
  3. From your app dashboard, you can find your App ID and App Secret.
  4. To generate an access token, go to the Graph API Explorer tool.
  5. Select your app from the dropdown and request the necessary permissions:
    • email - For email access
    • public_profile - For basic profile information
    • user_posts - To access your posts
    • user_photos - To access photos in posts
    • user_likes - To access reactions on posts
  6. Click "Generate Access Token" and follow the prompts.
  7. For a long-lived token, use the Access Token Debugger tool to extend its validity.

Important: To display posts, reactions, and comments, make sure your access token has the user_posts and user_likes permissions. Without these permissions, the dashboard will only show your basic profile information.
